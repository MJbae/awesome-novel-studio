# 장르 DNA 프레임워크 — 메타 설계 문서

이 문서는 장르 DNA가 **무엇을 포함해야 하는지**를 정의하는 메타 프레임워크이다.
실제 장르별 콘텐츠는 domain-researcher 에이전트가 이 프레임워크를 기반으로 동적으로 생성한다.

> **출력 위치**: `_workspace/00_research/genre-dna-{장르명}.md`

---

## 분석축 1: 핵심 카타르시스 구조

이 장르가 독자에게 쾌감을 주는 근본 메커니즘을 정의한다.

### 리서치 대상
- 해당 장르 상위 인기작 20개의 독자 리뷰 키워드 분석
- 독자가 "재미있다"고 느끼는 순간의 공통 패턴
- 장르 특유의 감정 루프 (긴장→해소, 미스터리→해결, 결핍→충족 등)
- 이중/다중 카타르시스 구조 여부 (예: 전문 지식 쾌감 + 복수 쾌감)

### 포함 내용
- **핵심 쾌감 정의**: 한 문장으로 요약 (예: "정보 우위를 통한 역전의 쾌감")
- **카타르시스 레이어**: 장르가 제공하는 쾌감의 층위 (최소 2개)
- **도파민 트리거**: 독자의 도파민을 유발하는 구체적 장면 유형 목록
- **리액션 공식**: 주변 인물/적대자의 반응이 쾌감에 기여하는 방식

### 출력 구조
```markdown
## 핵심 카타르시스 구조
### 핵심 쾌감 정의
[한 줄 요약]
### 카타르시스 레이어
| 레이어 | 설명 | 예시 |
### 도파민 트리거
1. [유형]: [설명]
### 리액션 공식
[주변 인물 반응 → 적대자 반응 → 독자 감정 패턴]
```

## 분석축 2: 서사 공식

검증된 에피소드별/전체 서사 패턴을 정의한다.

### 리서치 대상
- 상위 인기작의 초반 3~5화 구조 분석
- 25화/50화/100화 단위 아크 전환 패턴
- 전체 서사 구조의 공통 골격 (기승전결, 영웅서사, 성장서사 등)
- 유료 전환 시점 전후의 플롯 강화 패턴
- 장기 연재(200화+)에서의 소재 고갈 방지 전략

### 포함 내용
- **화수별 공식**: 1화, 2화, 3화, ~25화, ~50화, ~100화, 이후 구간별 핵심 내용
- **전체 서사 구조**: 시작→전개→위기→절정→결말의 장르 특화 변형
- **아크 구조**: 50화 단위 아크의 구성 원칙
- **서사 기점 설계**: 이야기의 시작점(회귀, 전생, 각성, 발견 등)을 설계하는 원칙
- **감정적 동인 설계**: 프롤로그/초반에서 주인공의 행동 동기를 극대화하는 패턴

### 출력 구조
```markdown
## 서사 공식
### 화수별 공식
| 구간 | 핵심 내용 |
### 전체 서사 구조
[시작 → ... → 결말]
### 서사 기점 설계
[장르 특화 시작점 패턴]
### 감정적 동인 설계
[독자 감정이입을 극대화하는 초반 설계 원칙]
```

## 분석축 3: 캐릭터 아키타입

해당 장르에서 성공하는 캐릭터 유형을 정의한다.

### 리서치 대상
- 인기작 주인공의 공통 특성 (성격, 능력, 동기, 매력 포인트)
- 조력자 유형 분류와 등장 리듬
- 적대자(빌런) 설계 패턴과 계층 구조
- 서브 캐릭터(로맨스/동료) 처리 방식
- 실패한 캐릭터 설계 사례 분석

### 포함 내용
- **주인공 아키타입**: 핵심 특성, 매력 공식, 성장 곡선
- **조력자 프레임워크**: 유형 분류, 등장 리듬, 역할 정의
- **적대자 프레임워크**: 계층 구조, 진화 패턴, 설계 원칙
- **핵심 인물 등장 리듬**: VIP/핵심 조력자의 등장 주기와 패턴
- **캐릭터 금기 사항**: 해당 장르에서 절대 해서는 안 되는 캐릭터 처리

### 출력 구조
```markdown
## 캐릭터 아키타입
### 주인공 아키타입
- 공식: [핵심 특성 조합]
- 매력 포인트: [목록]
### 조력자 프레임워크
| 유형 | 역할 | 등장 시기 |
### 적대자 프레임워크
| 계층 | 특성 | 등장/퇴장 리듬 |
### 핵심 인물 등장 리듬
[주기와 패턴]
### 캐릭터 금기
[절대 회피 목록]
```

## 분석축 4: 독자 내성/기대

독자가 참을 수 없는 것과 반드시 기대하는 것을 정의한다.

### 리서치 대상
- 독자 리뷰에서 부정적 반응을 유발한 패턴
- 독자 리뷰에서 긍정적 반응을 유발한 패턴
- 이탈 원인 분석 (어느 시점에 왜 떠나는가)
- 장르 독자층의 연령/성별/취향 프로파일
- 커뮤니티(리뷰, 댓글, 포럼)에서 반복되는 불만/칭찬

### 포함 내용
- **절대 금기**: 독자가 절대 수용하지 않는 패턴 (이유 포함)
- **필수 요소**: 독자가 반드시 기대하는 요소 (부재 시 이탈)
- **내성 수준**: 긴장/답답함/불쾌함에 대한 독자의 허용 범위
- **이탈 트리거**: 독자 이탈을 유발하는 구체적 패턴
- **충성 트리거**: 독자를 장기 구독자로 전환하는 패턴

### 출력 구조
```markdown
## 독자 내성/기대
### 절대 금기
| 패턴 | 이유 | 이탈 위험도 |
### 필수 요소
| 요소 | 부재 시 결과 |
### 내성 수준
| 감정 유형 | 허용 범위 |
### 이탈/충성 트리거
[구체적 패턴 목록]
```

## 분석축 5: 플랫폼 최적화

타겟 플랫폼의 독자 행동 패턴에 맞춘 최적화 전략을 정의한다.

### 리서치 대상
- 플랫폼별 독자 행동 패턴 (읽기 습관, 결제 패턴, 리뷰 문화)
- 플랫폼별 유료 전환 전략 (타이밍, 구간, 트리거)
- 플랫폼별 노출 알고리즘 특성
- 플랫폼별 추천 회차 분량 및 집필 구조
- 플랫폼별 성공 지표와 핵심 KPI

### 포함 내용
- **플랫폼 특성**: 독자층, 분위기, 트렌드 민감도
- **유료 전환 전략**: 최적 타이밍, 전환 직전/직후 플롯 강화 전략
- **집필 구조**: 회차당 분량, 서론/본론/결론 배분, 연재 주기
- **노출/추천 최적화**: 알고리즘에 유리한 패턴
- **핵심 KPI**: 성공을 측정하는 지표와 목표 수치

### 출력 구조
```markdown
## 플랫폼 최적화
### 플랫폼 특성
[독자층, 분위기, 핵심 지표]
### 유료 전환 전략
[타이밍, 전략, 플롯 강화 패턴]
### 집필 구조
| 요소 | 기준 |
### 노출/추천 최적화
[알고리즘 친화 전략]
```

## 분석축 6: 필수 회피 패턴

장르에서 흔히 실패하는 패턴을 정의한다.

### 리서치 대상
- 중단/완결 실패작의 공통 패턴 분석
- 독자 이탈이 집중된 구간과 원인
- 장르 특유의 함정 (클리셰 과다, 설정 파괴, 소재 고갈 등)
- 초보 작가가 반복하는 실수 패턴
- 인기 하락 시점과 원인 분석

### 포함 내용
- **치명적 실패 패턴**: 즉시 이탈을 유발하는 패턴 (예시 포함)
- **점진적 악화 패턴**: 서서히 흥미를 잃게 만드는 패턴
- **구조적 함정**: 장기 연재에서 빠지기 쉬운 구조적 문제
- **회피 전략**: 각 패턴에 대한 구체적 방지 방법

### 출력 구조
```markdown
## 필수 회피 패턴
| 패턴 | 유형 | 이유 | 회피 전략 |
```

## 분석축 7: 차별화 전략

레드오션에서 니치를 찾는 전략을 정의한다.

### 리서치 대상
- 해당 장르의 포화도 분석 (어떤 소재가 과포화인가)
- 성공적 차별화 사례 분석 (기존 공식에 무엇을 더했는가)
- 미개척 소재/설정/톤 영역 탐색
- 이종 장르 크로스오버 성공 사례
- 최신 트렌드와 향후 예측

### 포함 내용
- **포화 영역**: 이미 과포화된 소재/설정 목록
- **미개척 영역**: 아직 시도되지 않았거나 덜 시도된 소재/설정
- **성공적 변주 사례**: 검증된 차별화 방법 (사례 포함)
- **톤/무드 변주**: 기존 장르 톤에서 벗어나는 가능한 방향
- **크로스오버 가능성**: 이종 장르 결합 시 유효한 조합과 주의점

### 출력 구조
```markdown
## 차별화 전략
### 포화/미개척 영역
| 영역 | 상태 | 기회 |
### 성공적 변주 사례
[사례별 분석]
### 톤/무드 변주
[가능한 방향과 리스크]
### 크로스오버 가능성
| 조합 | 시너지 | 주의점 |
```

## 분석축 8: 장르 조합 분석

두 개 이상의 장르 DNA를 결합할 때의 원칙을 정의한다.

### 리서치 대상
- 성공적 장르 결합 사례 (어떤 요소를 주/부로 배분했는가)
- 실패한 장르 결합 사례 (왜 독자가 혼란을 느꼈는가)
- 장르별 핵심 요소의 호환성 매트릭스
- 결합 시 독자 기대 관리 전략

### 포함 내용
- **주장르/부장르 원칙**: 어느 장르의 카타르시스를 메인으로 할 것인가
- **호환성 매트릭스**: 장르 요소별 결합 가능 여부와 충돌 지점
- **결합 전략**: 서사/캐릭터/세계관 층위별 결합 방법
- **독자 기대 관리**: 장르 결합 시 독자 혼란을 방지하는 신호 설계
- **비율 조절**: 장르 요소의 배분 비율 가이드라인

### 출력 구조
```markdown
## 장르 조합 분석
### 주장르/부장르 원칙
[메인 카타르시스 결정 기준]
### 호환성 매트릭스
| 요소 A | 요소 B | 호환성 | 주의점 |
### 결합 전략
[서사/캐릭터/세계관별 결합 방법]
### 독자 기대 관리
[혼란 방지 전략]
```

## domain-researcher 에이전트를 위한 실행 가이드

### 리서치 프로세스
1. 사용자가 지정한 장르 키워드를 기반으로 WebSearch 실행
2. 8개 분석축 각각에 대해 최소 5개 이상의 출처에서 정보 수집
3. 수집된 정보를 분석축별 출력 구조에 맞게 정리
4. 상호 참조를 통해 모순/중복 제거
5. `_workspace/00_research/genre-dna-{장르명}.md`에 저장

### 품질 기준
- 모든 분석축에 구체적 예시가 1개 이상 포함되어야 한다
- 인기작 레퍼런스는 작품명과 구체적 화수를 명시한다
- "~인 것 같다" 식의 추측은 금지. 근거 기반으로 서술한다
- 독자 리뷰/커뮤니티 반응은 출처를 명시한다

### 업데이트 주기
- 장르 트렌드는 빠르게 변한다. 3개월마다 주요 분석축 재검증을 권장한다
- 새 인기작이 등장하면 해당 작품의 패턴을 기존 DNA에 반영한다
